Frequently Asked Questions

Find answers to common questions about TinyPixels image optimization

Image Compression

How does TinyPixels compress images?

We use advanced compression algorithms that analyze image content to remove redundant data while preserving visual quality. Our smart compression:

  • Optimizes color palettes
  • Removes unnecessary metadata
  • Applies content-aware compression
  • Uses modern formats like WebP when possible

What's the maximum file size I can process?

Since all processing happens in your browser, the limit depends on your device's memory. Generally, we recommend:

  • Individual files up to 50MB
  • Total batch size up to 500MB
  • Up to 100 images per batch

Privacy & Security

Are my images safe?

Absolutely! TinyPixels processes everything locally in your browser:

  • No server uploads required
  • Images never leave your device
  • No data collection or storage
  • Open-source processing algorithms

Do you store any user data?

No. We only use essential cookies for site functionality. We don't:

  • Store your images
  • Track processing history
  • Require user accounts
  • Share any information with third parties

File Formats

Which formats are supported?

We support all major image formats:

  • Input: JPEG, PNG, WebP, GIF, BMP, TIFF
  • Output: JPEG, PNG, WebP, AVIF, HEIC
  • Animated GIF support
  • RAW file conversion

Which format should I choose?

Recommended formats for different uses:

  • Photos: JPEG or WebP
  • Logos/Graphics: PNG or SVG
  • Web Images: WebP with JPEG fallback
  • Modern Browsers: AVIF for best compression

Usage & Features

How do I batch process images?

Batch processing is simple:

  • Drag multiple files into the dropzone
  • Set desired options once for all images
  • Click "Process Images"
  • Download individually or as ZIP

What are the quality settings?

Our quality settings are optimized for different needs:

  • Maximum (100%): Lossless compression
  • High (80%): Recommended for photos
  • Medium (60%): Good for web images
  • Low (40%): Maximum compression

Mobile & Tablets

Can I use TinyPixels on mobile devices?

Yes, TinyPixels is fully optimized for mobile use:

  • Responsive touch-friendly interface
  • Works with mobile browsers
  • Access camera roll directly
  • Optimized for low bandwidth

How can I optimize images for mobile viewing?

For best mobile performance:

  • Use WebP format when possible
  • Set width to common mobile breakpoints (360px, 414px, etc.)
  • Enable responsive image sizing
  • Keep file sizes under 200KB for fast loading
